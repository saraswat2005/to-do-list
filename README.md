# 📝 To-Do List

A simple To-Do List application built using **HTML, CSS, and JavaScript**. It allows users to add tasks, mark them as completed, and delete them.

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🎨 Simple and clean user interface
- ⚡ Built using Vanilla JavaScript (No libraries)

## 📂 Project Structure

```
📁 To-Do-List
│── index.html
│── 10.js
│── README.md
```

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 📖 How It Works

1. Enter a task in the input field.
2. Click the **ADD** button.
3. The task appears below the input field.
4. Click **Done** to mark it as completed.
5. Click **Delete** to remove the task.
